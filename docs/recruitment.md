# LAS 文学分析系统 — 国赛团队招募

## 我们在做什么

LAS（Literary Analysis System）是一套基于大语言模型的十六维文学价值评估系统。简单说：**让 AI 像专业文学批评家一样，对任意一部作品给出有证据链的深度分析报告。**

目前系统已经是一个完整的 Web 应用——用户提交作品（经典名作只需输入书名，原创作品需粘贴正文），DeepSeek V4 大模型会从语言艺术性、叙事技巧、人物塑造、象征体系等 16 个维度逐一评分，输出包含雷达图、评分详表、深度分析和修改建议的交互式报告。分析过程通过 SSE 实时流式展示，用户体验流畅。

核心创新点：**每个分数背后都有具体文本证据和与经典作品的对比**——不是 AI 拍脑袋打分，而是规则驱动的结构化分析。

## 项目现状

- **版本**：v5.2.3，全链路通畅，无已知 Bug
- **技术栈**：FastAPI + Vanilla JS SPA + DeepSeek V4 API + SQLite + Chart.js
- **已完成**：271 部作品的系统测评（计划 565 部），覆盖东亚/欧洲/北美/拉美等 10 个区域，评分从《离骚》137 分到当代习作 32 分
- **评分框架**：16 维锚定标尺 + 体裁自适应权重 + 原创审慎校验 + 偏科/平庸双重惩罚机制
- **提示词**：58KB 系统提示词（6 公理 + 13 步工作流 + 15 级锚定表），已迭代至 V2 JSON 模式
- **外部合作**：南开大学电子信息学院技术合作已达成；本校教师初步反馈已获得
- **启动资金**：2000 元已自筹到位
